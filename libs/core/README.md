# `@tanstack-isomorphic-form/core`

Core primitives for `@tanstack-isomorphic-form`.

This package provides the shared form loader, form-data extraction, and action helpers.

## Installation

```bash
pnpm add @tanstack-isomorphic-form/core
```

## Exports

- `createFormDataExtractor`
- `createFormLoader`
- `runFormAction`
- `redirectAfterAction`
- `FormActionPanic`
